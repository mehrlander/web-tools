Compress it. Then prove it round-trips.

window.compression wraps Brotli (WASM), Gzip (CompressionStream), acorn JS detection, and a bookmarklet/data-URL packer. Each snippet runs in an isolated frame with the kit injected — edit it, press Run, read the console.

Colophon

Driven by demo.mount(…) from vanilla-demo.js with the editor from kits/cm6.js. Each block declares inject: ['kits/compression.js']; the proof frame loads it as a plain script before the snippet runs.

The full UI that composes these calls is compression-helper.html.
