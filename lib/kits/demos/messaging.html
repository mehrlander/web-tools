window.messaging is a tiny exact-match pub/sub bus that lets detached components coordinate without a shared store. Each snippet runs in its own isolated frame with the kit injected — edit it, press Run, read the console.

Driven by demo.mount(…) from vanilla-demo.js. Each block injects kits/messaging.js into its frame; subscriptions are per-frame, so blocks can't interfere.
