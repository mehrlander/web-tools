kits / demos

web-tools

Small pages that exercise one kits/ module each. The integration that uses every kit together lives in pages/compression-helper.html.

  • persistence String-path key/value over idb-keyval with structured-clone payloads.
  • messaging In-memory pub/sub keyed on opaque path strings; lets detached components coordinate.
  • io File pick/save, blob preview, clipboard. Lazy JSZip for saveZip.
  • export Export a page plus the data it read()s as one zip, laid out so unzip-and-open resolves local. Optionally bakes the code in for a fully-offline page.
  • compression Brotli/Gzip + acorn + bookmarklet packer. Full UI: compression-helper.html.
  • selfsim A catalogue of text-analysis functions over a suffix automaton: exact repeated passages, occurrence counts, longest repeat, and a one-call dot-plot renderer.
