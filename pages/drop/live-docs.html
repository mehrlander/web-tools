A concept · Living documentation · daisyUI

Show the code. Then prove it.

A page is a stack of sections; each section gathers a few examples. Every example pairs prose with a snippet you can edit and a proof that follows in the same card: the markup rendering, the same markup in its habitat, or the console answering back.

Colophon

Driven by demo.mount({ sections, sectionsEl, displayEl, legendEl }) from vanilla-demo.js, with the editor from kits/cm6.js. Each example is one config object; only the section carries a large heading.

The proof is an isolated iframe, so a snippet cannot reach the page around it. Display preferences, pinned top-right, apply to every editor at once.
