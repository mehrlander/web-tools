alpine-bundle directives

web-tools

Directives registered in alpine-bundle.js. Each decorates the host element — adds classes, wires behavior, or renders reactive content — based on dot-modifiers parsed from the attribute name. The class-shortcut tier mirrors helpers in kits/fills.js; the reactive and behavior tiers are new.

x-btn — class shortcut

x-btn.xs.primary="count++" → adds btn btn-xs btn-primary and wires the expression as a click handler. Unknown modifiers pass through as raw classes, so x-btn.xs.shadow-md works too.

x-tip.top="'<b>rich</b> body'" → wraps the element as a daisyUI tooltip; modifier picks the position; expression is the body (HTML allowed).

x-metric — formatter + reactive

x-metric.currency.trend="{ label, val, prev }" → renders a metric card with Intl-formatted value and a trend indicator. Modifiers shape the rendering pipeline: .currency / .percent for formatting, .trend to enable the diff arrow, .invert to swap the favorable direction.

x-action.confirm — behavior wrapper

x-action.confirm="..." → confirmation-gated click. First tap arms (turns red, swaps label to "Are you sure?"); second tap within 3s executes; otherwise reverts. Without .confirm it's a plain click.
