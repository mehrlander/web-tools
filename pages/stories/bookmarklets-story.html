01 / The Wall

CSP, or: why your clever idea fails on the one site you actually need

A bookmarklet injecting <script src="cdn..."> works beautifully on permissive pages and fails silently on GitHub, Notion, and most enterprise tools. The CSP header is doing exactly its job. The practical rule has two parts: bundle everything inline, and if you need a remote script, open a temporary window that operates outside the parent page's CSP entirely. That's the Lambdalet approach. It's the most underrated trick in this space.

03 / The Stack

The pipeline that almost exists

Gitingest and Repomix flatten repos into LLM-ready artifacts — both have MCP servers, meaning Claude can trigger repo packing as a tool call. give-me/bookmarklets exports conversations across all major platforms. The GitHub API commit pattern — stored PAT, extract title and URL, POST a named markdown file — is proven, just never bundled cleanly. What doesn't exist: a unified "capture with provenance" tool that shows what it's extracting and why. And the bookmarklet-to-MCP bridge — completely unexplored — which would invert the model from "human captures, Claude reads later" to "Claude decides when to capture."

04 / The Sources

Five AIs walked into a research prompt

The measure that actually matters is net-new signal, not word count. ChatGPT found Gitingest and the best CSP analysis. Grok found the GitHub API commit pattern and the MCP-exposure idea. DeepSeek named the launcher pattern cleanly. Gemini buried its real findings in academic filler — then invented Paul Ford's opinions on AI agents in 2026 and presented them as sourced quotes. That's not extrapolation. That's confabulation dressed as research. The lesson: deep research tools do wide well. Depth requires judgment — knowing what matters, what's uncertain, and when to stop.
