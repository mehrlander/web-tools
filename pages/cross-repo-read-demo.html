Repo data → local file. One read(), two sources.

read(path) resolves a path locally if a file sits beside the page, otherwise from the repo. Read the repo copy here, then Save local copy to freeze it into a document.currentScript.value file. How it resolves

local — read injects <script src=path>; the file deposits its payload on document.currentScript.value and read lifts it off the node. The only thing that runs on file://.

repo — if there's no local file, read fetches via the API and runs the file with gh injected; the file just returns its payload. Neither path touches window.
