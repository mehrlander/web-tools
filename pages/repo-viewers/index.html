repo viewers

web-tools

Pages that browse a GitHub repository's files. They share the gh-fetch.js loader and the Alpine repo/navigator/viewer components.

  • show-repo-assembled Leading-edge GH browser. Uses the *-assembled.js components and a boot-error fallback.
  • show-repo Original GH browser, on the older non-assembled repo.js/navigator.js/viewer.js.
  • repo-drag Drag-and-drop repo file viewer.
  • demo-viewer Smallest demo of viewer-assembled.js, picking files from this repo.
