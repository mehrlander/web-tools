Reading the map
Every file is a tile — area is its share of the view (bytes, or file count via the area toggle). Color is the file's type; directories are the darker frames.
Getting around
click dirzoom in (breadcrumbs / Backspace to go up)
click fileselect, details in the sidebar
dbl-clickopen the file on GitHub
type rowshover to preview, click to isolate a type
Escclear search / filter / selection
Any public repo works — paste owner/name above. For private repos or API rate limits, store a token as localStorage.ghToken. The URL tracks the view, so it's shareable.
Local files map too: drop a folder or .zip on the map, or use the picker. Nothing uploads — it all stays in this tab.
