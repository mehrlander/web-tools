Popup Launcher

One bookmarklet opens the popup menu on whatever page you're on. Paste a GitHub token below and copy out a bookmarklet to save. On a public repo the token is optional — leave it blank and it works at the unauthenticated rate limit.

A fine-grained PAT with read-only Contents on this one repo is plenty. The token gets baked into the bookmarklet in plaintext, so it lives wherever you save the bookmark — use a narrowly-scoped token.

How to install
  • Desktop: drag the Open Popups button to your bookmarks bar.
  • Mobile: tap Copy, bookmark any page, then edit that bookmark and replace its URL with what you copied. (iOS/Android hide the bookmarks bar, so this is the reliable path.)
  • Run it on any page. A window opens with the popup menu; pick a tool and it loads in place, coupled to that page's origin.

These links open each popup standalone on GitHub Pages — useful to preview, but that runs against this site's own origin (a bench, not the job). The bookmarklet is how you run them coupled to a real host page.
