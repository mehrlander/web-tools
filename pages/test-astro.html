A short lesson on what a build step is, rendered by a build step.

Astro read this JSON

These cards started life as a JSON file. The build process read it, looped over the entries, and stamped each one into the HTML before you arrived.

Tailwind compiled to CSS

The classes on every element got scanned, matched against the Tailwind config, and turned into a small stylesheet. None of that work happens in your browser.

Components flattened into HTML

The source has a Layout, an index page, and these card components. The output is one HTML file with no trace of the composition. The seams disappeared at build time.

What you see is what shipped

Open the page source. The text of this card is right there as a string. Nothing fetched it, nothing rendered it on the fly. It was decided in advance.
