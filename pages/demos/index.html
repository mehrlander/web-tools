kits / demos

web-tools

Small pages that exercise one kits/ module each. The integration test that uses every kit together is compression-helper-v2.html, sitting next to the still-working compression-helper.html.

  • fills daisyUI/Tailwind template-string helpers (tip, lines, btn, toolbar, modal).
  • persistence String-path key/value over idb-keyval with structured-clone payloads.
  • messaging In-memory pub/sub keyed on opaque path strings; lets detached components coordinate.
  • io File pick/save, blob preview, clipboard. Lazy JSZip for saveZip.
  • component Thin defineComponent(name, tpl, data) over Alpine + custom elements.
  • compression Brotli/Gzip + acorn + bookmarklet packer. Live integration: compression-helper-v2.html.
