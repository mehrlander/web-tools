demos

Pure HTML template helpers. Each block below shows one helper, its source expression, and the rendered result. From window.fills (source).

tip(mods, trigger, content)

daisyUI tooltip with rich HTML body. mods takes a position (top|bottom|left|right) and any text mods (xs|sm|bold|...).

lines(mods, arr)

Stacked column. mods takes a gap (gap-0..4) and text mods.

btn(mods, label, click, iconClasses?, extraClasses?)

daisyUI button. mods carries size + variant (xs|sm, primary|ghost|...). iconClasses is a phosphor class.

toolbar(mods, ...items)

Horizontal flex wrapper. Items are HTML strings; usually nested fills.

modal(inner)

Full-screen daisyUI dialog. Open via the dialog's showModal().

saveIndicator()

Spinner shown while saving is truthy in the surrounding x-data. Toggle below to fake a save.

pathInput()

Edits a _path field; commits to path on blur/Enter. Designed to pair with kits/persistence.js (next demo). Here it just round-trips strings.
