demos

window.component.defineComponent(name, tpl, data) registers a custom element backed by an Alpine x-data factory. Once defined, instances appear in markup as a tag — <my-thing attr="…"> — with attributes available to both the template function and the data factory. After Alpine binds, the reactive data object is stashed on the host as host.__data, and an optional onMount(host) on that object runs once. (kit source)

This sits alongside the other path the repo uses for reusable Alpine pieces: files under alpineComponents/ that call Alpine.data('name', fn) inside an alpine:init listener, mounted in the page by writing <div x-data="name()">…</div>. Both paths register the same shape of Alpine factory; what differs is the surface — a custom-element tag vs. an x-data attribute — and where the template HTML lives (passed to defineComponent vs. written in the page body).

Demos below: a self-contained counter, attribute passing, two components coordinating via kits/messaging.js, and reaching into a bound instance from outside.

1. Self-contained counter

defineComponent('demo-counter',
  () => `<button class="btn btn-sm" @click="inc()" x-text="`Count: ${count}`"></button>`,
  () => ({ count: 0, inc() { this.count++ } })
);
<demo-counter></demo-counter>

2. Reading attributes

defineComponent('demo-greeting',
  attrs => `<div>Hello, <b x-text="name"></b>! (passed start=${attrs.start ?? 0})</div>`,
  attrs => ({ name: attrs.name ?? 'world', start: Number(attrs.start ?? 0) })
);
<demo-greeting name="Ada" start="42"></demo-greeting>

3. Cross-component coordination via messaging

The slider publishes on demo.knob; the gauge subscribes. Neither holds a reference to the other. 

defineComponent('demo-knob',
  () => `<input type="range" min="0" max="100" x-model.number="v">`,
  () => ({ v: 50, onMount() { this.$watch('v', n => messaging.publish('demo.knob', 'change', n)); } })
);
defineComponent('demo-gauge',
  () => `<div class="text-3xl font-bold" x-text="n"></div>`,
  () => ({ n: 0, off: null, onMount() { this.off = messaging.subscribe('demo.knob', (_, n) => this.n = n); } })
);
(updates as you drag)
