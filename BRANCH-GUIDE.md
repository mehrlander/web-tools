**What's new (all additive; existing `#gh=`/`#gz=`/`#html=`/`#url=` and the token/sandbox model unchanged):**
- "Paste & render" button (Clipboard API, one-tap on iOS, text never shown) + a global `paste` listener. HTML → sandboxed; a bare URL → `#url`; a `github.com/.../blob/...` URL → trusted `#gh=`.
- On-page GitHub-file field: `owner/repo[@ref]:path` or a blob URL → `showAddress()` (private repos via the same `localStorage.ghToken` show-repo uses; `OWNERS=['mehrlander']` allowlist).
- "Copy toss link" pill: rebuilds the durable `#gz=`/`#url=`/`#gh=` bookmark for whatever is on screen.
- FAB (`alpineComponents/fab.js`) mounted for zip/offline export + its tabs. Boots after the render logic and self-loads its kits, so a blocked CDN degrades to render-only.
