# Branch guide: claude/hex-nut-icon-sizing-jprpnk

Enlarges the web-tools hex-nut favicon so it fills the tab slot instead of rendering small, by tightening the SVG `viewBox` to frame the artwork.

⭐ [pages/index.html](https://mehrlander.github.io/web-tools/pages/index.html?use=claude/hex-nut-icon-sizing-jprpnk)

**Cause:** the hexagon path spans x 5-27 (22 of 32 units) and y 6.474-25.526 (19 of 32), so ~30% of the canvas width and ~40% of the height was empty margin. Browsers scale the whole 32x32 viewBox into the ~16px tab slot, so the mark landed ~11px wide. Fix reframes the viewBox to a square box centered on (16,16), side 23, leaving path and mask coordinates untouched (mask rects are `userSpaceOnUse`, so they stay aligned).
