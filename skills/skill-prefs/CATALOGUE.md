# Skill Library Catalogue

The standing map: family, type, and premise per skill. Consult when placing or naming a new skill, or checking its neighbors. Update when a skill is born, renamed, or retired. Bundled with skill-prefs; the unit of update is the folder.

Type marks how a skill is used: reference (read and apply), method (a way of working), generator (produces a fixed artifact), flag (marks session posture). ⍰ marks a family call still open.
