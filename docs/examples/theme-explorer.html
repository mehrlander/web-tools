Vendor & inject

Every library on this page loads from its own CDN tag: Tailwind, daisyUI, Phosphor and Tabulator from jsDelivr, Alpine from unpkg. Behind a locked-down allowlist those hosts are blocked, so a headless render harness vendors each package from npm and injects it in place of the CDN response. The HTML never changes: same URLs, different source. daisyUI's themes.css is fetched the same way, then parsed into the table above, so the matrix is the library's own source, not a transcription.
