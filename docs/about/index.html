About

This repo holds an Astro project. The source lives in /src, the compiled site lives in /docs, and both are committed so the pre-build and post-build content live side by side.

To change something, edit any file under /src via the GitHub web UI, then ask Claude Code to pull the repo and rebuild. It runs npm run build in its sandbox and commits the updated /docs.

Never edit /docs by hand. It is generated output, and the next build will quietly overwrite your changes.
